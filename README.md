## Getting Started

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### Installation
> - `git clone `
>
> - `cd /`    
>
> - `npm install` to install all dependencies  
>
> - `npm start` to start your react web server   

### Tech Stack 
> - create-react-app
>
> - React with Hooks
>
> - NodeJS
>
> - Socket.io
